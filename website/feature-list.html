Any features tagged with a * are required for the minimum viable product.

  • The ability to create a new collection*
  • The ability to remove a chosen collection*
  • The ability to add “items” to a chosen collection*
  • The ability to remove items from a collection*
  • The ability to save your collections*
  • The ability to name a collection
  • The ability to name items in a collection
  • The ability to add a description to items in a collection
  • The ability to add images to items in a collection
  • The ability to create groups/categories in a collection
  • The ability to nest collections in other collections
  • The ability to add notes to items
  • The ability to add tags to items in a collection
  • The ability to search keywords in a collection
  • The ability to sort by tags
  • The ability to select multiple items and perform mass edits
  • The ability to upload/share your collection
  • The ability to view other peoples collection
  • The ability to add a wishlist to a collection
  • Show number of items inside a collection
  • Show amount of a single item in a collection
  • Access to sites that allow you to check value of items
  • The ability to print labels to stick on your physical items.
